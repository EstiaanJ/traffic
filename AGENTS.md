# Project Details
This project is a machine learning project where the environment is a 2d top down driving simulator created in Godot engine and the AI side is written in Java.
- Godot 4.5 (Ensure code is compatabile with Godot 4 as it has major changes from Godot 3)
- Java 21

## Style & Principles
### Java
Use functional programming where possible, immutable types, multi-thread safety and Java records.
Use composition over inheritance
